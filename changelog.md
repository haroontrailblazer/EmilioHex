##  [v1.0] – First Release
### Added
- Basic EMI calculation (principal, interest, total payable)  
- Simple interest-based EMI logic  
- Matplotlib pie chart showing:
  - Principal amount  
  - Total interest  
  - Monthly payment  
- Clean single-page Streamlit UI  
- Auto form hide/show on calculation  
- Footer with contact links  
- Dark theme styling using HTML/CSS  
- Minimal dependency setup (Streamlit + Matplotlib)  
- Dockerfile for deployment  

### Notes
- This version represents the **first ever build** of Emilio  
- Created as a personal tool to understand laptop EMI payments  
- Lightweight and beginner-friendly
